# DCM Framework — Data Repository

This folder contains the key output datasets from the 
Edendale–Southland proof of concept, enabling 
independent verification of all findings reported in 
Module 6.

| File | Description | Module reference |
|---|---|---|
| rdm_summary_2035_EB.csv | RDM screening results for EB pathway, 100 futures | §6.9 |
| rdm_summary_2035_BB.csv | RDM screening results for BB pathway, 100 futures | §6.9 |
| compare_2035_EB_vs_BB.csv | Site-layer pathway comparison | §6.9 |

These files are governed outputs under the DCM 
artefact schema. Each row corresponds to one future 
in the paired-futures ensemble. See SM-6.2-A for 
the full pipeline documentation.
