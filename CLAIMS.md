# DCM Framework — Claims Register

This register lists the primary technical claims of
the Decision-Centred Modelling framework that are
open for domain expert attestation.

Submit attestations via GitHub Issues using the
template provided in CONTRIBUTING.md.

## Status vocabulary

- ✓ Confirmed — attested by named domain expert
- ≈ Confirmed with qualification — attested with caveat
- ✗ Contested — disputed with documented basis
- ○ Open — awaiting attestation
