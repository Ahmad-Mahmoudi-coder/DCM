# A Decision-Centred Modelling Environment

**A Decision-Centred Modelling Environment for Long-Horizon Planning 
Under Deep Uncertainty: Foundations, Architecture, and Application to 
Industrial Process Heat Decarbonisation**

*Seyed Ahmad Mahmoudi Lahijani | Auckland University of Technology | 2025*

---

## The Core Finding

When the Edendale electrification pathway is evaluated against a 
structured ensemble of 64 futures, **23 of those futures** produce grid 
hosting capacity exceedances that generate regional infrastructure costs 
invisible from within the site boundary alone.

This framework makes those consequences visible — and attributable.

---

## What This Is

This repository contains:

- The full manuscript of the framework (in `manuscript/`)
- The proof-of-concept pipeline documentation (in `manuscript/sub-modules/`)
- A Quarto book project that renders to HTML website, PDF preprint, 
  and Word document from the same source

The framework's central commitment:

> *The decision makes the boundary; the artefact makes the connection.*

## How to Build

Install [Quarto](https://quarto.org/docs/get-started/) then run:

```bash
# Render all formats
quarto render

# Render HTML only (fastest for drafting)
quarto render --to html

# Preview with live reload
quarto preview
```
